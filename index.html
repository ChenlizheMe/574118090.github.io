Vultorch

Real-time CUDA tensor visualization with Vulkan zero-copy. show() does GPU→GPU blit; create_tensor() eliminates memcpy entirely via shared memory. Built-in ImGui, 3D scene view, docking layout. 基于Vulkan零拷贝的CUDA张量实时可视化工具。show() 实现GPU→GPU直传；create_tensor() 通过共享显存完全消除内存拷贝。内置ImGui、3D场景视图与可拖拽窗口布局。

InfEngine

A research-oriented game engine built on Vulkan + C++17 with a full Python scripting layer via pybind11. Features RenderGraph-based pipeline, ECS architecture, and modular design for graphics research and indie dev. 面向科研的游戏引擎，基于Vulkan + C++17构建，通过pybind11提供完整Python脚本层。采用RenderGraph渲染管线、ECS架构，模块化设计兼顾图形学研究与独立开发。
